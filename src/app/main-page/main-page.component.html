Oblivion

Библиотека компонентов

Oblivion это библиотека построенная на основе angular 4 и bootstrap 4, предназначенная для быстрого прототипирования приложений.

1. Все компоненты независимы

Введу факта того что angular4 предоставляет возможность изолировать обьекты друг от друга включая их стилевое оформление, все компоненты полностью изолированы и могут быть использованы отдельно

2. Для применения вам потребуется бутстрап и ангуляр

bootstrap 4.0.0-alpha.6 и ангуляр 4.0.0
( вероятно будет работать начиная с 2.1)

3. Компоненты сконфигурированы

Вставляя новый компонент вы сразу получаете конечный результат, все сконфигурировано для вашего удобства, Описаны все входящие данные и исходящие события, весь код задокументирован и лего читаем.
