Введение

Ежедневно работая с компонентами, рано или поздно приходит мысль, а зачем я делаю это 100 раз, и каждый раз заново. Хочется повторно использовать код и не вспоминать как там это было, не искать старые проекты и не выдерать код из них.

Библиотека Oblivion это как раз библиотека для быстрого старта.

Чтобы старт был быстрым, в библиотеке собираются компоненты, годные для многократного использования, на данный момент по мимо ангуляра в библиотеку входит bootstrap, в ближайшее время компоненты будут изолированы, а стили бутстрапа будут выделены и внесены в конкретный файл компонента. На данный момент пока это не возможно и у вас должен быть подключен bootstrap.

Быстрый старт

Установите ангуляр cli и сам ангуляр 4, выполнив в командной строке следующие команды
npm install -g @angular/cli

Сгерерируйте свое приложение перейдите в него и запустите, выполнив в командной строке следующие команды
ng new PROJECT_NAME
cd PROJECT_NAME
ng serve --open

Установите бутстрап скачав его при помощи npm или bower смотрите тут

Скачайте zip архив с гитхаба или склонируйте его в свой проект вот по этой ссыле

Библиотека: https://github.com/Gordondalos/oblivion
Git: https://github.com/Gordondalos/oblivion.git

Надите нужный компонент в библиотеке скопируйте папку с компонентом в папку с вашим приложением, и подключите указанные в документации файлы, пропишите в конфиге url адреса, картинок если они используются в компоненте. Если y компонента реализованы ивенты реализуйте методы обрабатывающие их. Если затрудняетесь с использованием компонента то вы можете прочитать файл readme.md лежащий в каждом компоненте, в нем подробно описано как подключить и сконфигурировать компонент.
